Adds a Facebook Like button to Page, Post or Custom Post Type content, requiring it to be clicked in order to 
access the content.

== Description ==

WP Like Lock allows site owners to 'lock' content on any Page, Post or Custom Post Type, until the site visitor clicks
the Facebook Like button to then view the content.

Content can be locked in full or in part on a per-Post basis, in conjunction with global options to lock content
by Post, Page, Custom Post Types or Taxonomy.

== Installation ==

1. Download the Like Lock Plugin.
2. In your WordPress Administration, go to Plugins > Add New > Upload, and select the plugin ZIP file.
3. Activate the plugin
4. Use the Settings Panel to lock content as required
